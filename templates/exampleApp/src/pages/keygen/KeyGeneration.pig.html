A simple text-to-hash workflow using an API src/pages/keygen/KeyGeneration.pig.html

Call the hash api here

Users can enter any text into the input field and submit it. The value is then sent to an API server/api/crypto/controller.mjs, which returns a hashed version of the input. The hashed result is displayed in a read-only <textarea>. If the app is running on a secure origin (HTTPS or localhost), a copy-to-clipboard icon appears in the corner of textarea. Clicking this icon copies the hash using the navigator.clipboard API. The clipboard feature is automatically hidden when unavailable due to browser security restrictions.
