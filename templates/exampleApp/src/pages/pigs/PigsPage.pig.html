Here we have a simple example of a page src/pages/pigs/PigsPage.pig.html

It demonstrates how state flows down and events bubble up between components, highlighting reactivity, conditional logic, and parent-child communication in the framework. It contains two components:

1. State Holder & Conditional Renderer
src/components/PigsContainer/PigsContainer.pig.html

The main component is responsible for managing deep reactive state â€” meaning it automatically re-renders when any nested property is updated. It checks if all tracked values are even and conditionally renders a message based on that. Additionally, it displays a table showing the number of times each of the five pigs has been clicked.

2. Pig Image Renderer & Click Listener
src/components/ClickCounter/ClickCounter.pig.html

This subcomponent is embedded within the main component and is responsible for rendering images of the pigs with a map for click areas and notifying the parent component about clicks. It receives a callback from the parent component in passed attributes, allowing each pig image to handle click events and notify the parent about interactions.
